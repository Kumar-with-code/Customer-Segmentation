# 🚀 Customer Segmentation Web App (ML Project)

An end-to-end Machine Learning web application that segments customers based on **Annual Income** and **Spending Score** using **K-Means Clustering**.


## 📌 Project Overview
This project uses **K-Means Clustering** to group customers into meaningful categories such as:

- 💎 Premium Customers  
- 💰 High Income – Low Spending  
- 🛍️ Low Income – High Spending  
- 👥 Average Customers  
The trained model is deployed using a **Flask Web Application** with a clean and responsive **Tailwind CSS UI**.


## 🧠 Machine Learning Model
- **Algorithm:** K-Means Clustering  
- **Features Used:**
  - Annual Income
  - Spending Score
